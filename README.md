# Retrieval-Augmented Conversational AI Chatbot (RAG + LLM)

This project is a Retrieval-Augmented Generation (RAG) based chatbot that can answer user queries using information retrieved from uploaded documents.  
It combines document embeddings, vector search, and Large Language Models (LLMs) to generate accurate, context-aware responses.
